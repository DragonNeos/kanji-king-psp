Kanji King PSP

Digital flash cards for all Heisig Kanji and JLPT vocabulary.
Switch between 3 high-res TrueType fonts.
Keywords, Components and Vocabulary for each Kanji in English, German and Japanese, and on- and kun- readings in Katakana and Hiragana.
Automatic rating and progress.

Controls:

Main learning screen:
Circle.....known kanji
Cross......unknown kanji
Triangle...revise answer
Square.....goto kanji list screen
L..........switch font
R..........switch language
Select.....switch learning mode
Start......goto components screen

Kanji list screen:
D-Pad......scroll list
Circle.....goto vocab screen
Cross......goto components screen
Square.....return to main screen
Select.....clear progress

Components and Vocab screen:
D-Pad......scroll through entries
Circle.....goto vocab screen
Cross......goto components screen
Square.....return to kanji list screen
